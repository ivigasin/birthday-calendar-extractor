Созданный календарь можно сохранить в формате `ICS` или `CSV`.

Как пользоваться этим дополнением
Просто нажимайте на иконку дополнения, всплывающие подсказки помогут завершить процесс.
Каждое нажатие будет приближать вас к цели :)
В конце-концов, файл с именем `birthday-calendar.ics` будет сохранен автоматически в папку Downloads.
Используйте сгенерированный файл для экспорта дней рождений ваших друзей в программу календарь.

Click on extension icon and select on of required formats:
* Создать Google Calendar - ICS - универсальный формат календарей, используемый в Microsoft Outlook, Google Calendar, и Apple Calendar.
* Создать Excel file - CSV
* Удалить из Google Calendar - генерирует файл для удаления уже импортированных дней рождений из календаря.

Важно!
Я не советую импортировать сгенерированный календарь в уже существующий, лучше создать новый и использовать его только для дней рождений.

ЧАВО
В: Я случайно сохранил дни рождения в свой основной календарь, как мне их удалить или отменить?
О: Нажмите на иконку дополнения ("торт") и выберете "Удалить из Google Calendar", импортируйте созданный файл в ваш календарь.
